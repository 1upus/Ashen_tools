# Ashen_tools
Tools for N-Gage Ashen and Operation Shadow from Torus Games
repo: https://github.com/1upus/Ashen_tools/
-----------------------------------------------------------------------
Platform: 
Nokia N-Gage

Games:
+ Ashen
+ Operation Shadow

Ashen Tools by Yoti (https://github.com/Yoti)
-----------------------------------------------------------------------
Pack of tools for unpack/rebuild Packfile.dat
Extracted text encoded with UTF 16-LE with BOM. Notepad ++ is recommrnded for editing.

ngSpriteTool_v1.2.1 by Metlob (https://github.com/MetLob)
-----------------------------------------------------------------------
Toolkit for export/import GFX from Torus sprite pack.
Also works as Tinke plug-in
https://github.com/MetLob/tinke/releases
